**AI Placement Management System**

**Overview**

The AI Placement Management System is an animated single-page web application designed to simplify the campus placement process. It provides separate login interfaces for Students and Administrators, offering features that help students find suitable job opportunities while enabling administrators to monitor placement activities through an interactive dashboard.

**Features**

**Student Portal**
Secure student login interface.
Browse and apply for available job opportunities.
AI-inspired job recommendations based on student skills.
Resume ATS (Applicant Tracking System) Score Checker.
References to recommended courses for skill enhancement.
Modern animated and responsive user interface.

**Admin Portal**
Secure administrator login.
Dashboard displaying all applied students.
Interactive bar graph for placement/application statistics.
Overview of student applications.
Clean and user-friendly dashboard design.

**How to Run**

Download or clone the repository.
Open the project folder.
Double-click index.html or open it in any modern web browser.
Use the Student or Admin portal from the home page.

**Note**
This project is currently a frontend interface prototype.
Login functionality, job applications, ATS scoring, and dashboard data are demonstrated through the UI and are not connected to a backend database.
It can be integrated with technologies like FastAPI, Node.js, Django, Firebase, or MySQL for full-stack functionality.
